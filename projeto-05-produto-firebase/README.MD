## 📌 Sobre o Projeto

O **ABNGrid** é uma aplicação web desenvolvida com React + TypeScript focada na criação de QR Codes modernos e totalmente personalizáveis.

A plataforma permite alterar estilos visuais, cores, formatos dos pontos, adicionar logos e salvar QR Codes na nuvem utilizando Firebase.

O projeto foi desenvolvido utilizando o **Google AI Studio** com integração à API **Gemini AI**.
