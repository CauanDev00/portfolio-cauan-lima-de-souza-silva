# 📌 Sobre o Projeto

O **ABNGrid Analytics Dashboard** é uma aplicação web desenvolvida para monitoramento e análise de dados em tempo real através de dashboards dinâmicos e visualização estratégica de informações.

O projeto foi estruturado utilizando **React + TypeScript**, permitindo transformar dados operacionais em indicadores visuais para apoio à tomada de decisão.

A solução integra:

- dashboards interativos
- monitoramento de KPIs
- visualização de métricas
- análise de performance
- armazenamento em nuvem
- integração com IA generativa

---

# 🧠 Objetivo Técnico

O objetivo principal foi estruturar um sistema de visualização de dados capaz de:

- transformar datasets em insights acionáveis
- monitorar métricas críticas
- identificar padrões e anomalias
- melhorar análise operacional
- facilitar interpretação visual dos dados

O dashboard foi desenvolvido priorizando:

- escaneabilidade
- legibilidade
- atualização dinâmica
- organização hierárquica das informações

# ⚙️ Como Executar o Projeto

## 📋 Pré-requisitos

Antes de iniciar:

- Node.js 18+
- Git instalado
- Conta Firebase
- Editor VS Code recomendado

---

## 1️⃣ Clonar o repositório

```bash
git clone https://github.com/CauanDev00/portfolio-cauan-lima-de-souza-silva
```

---

## 2️⃣ Entrar na pasta do projeto

```bash
cd projeto-05-dashboard-analitico
```

---

## 3️⃣ Instalar dependências

```bash
npm install
```

---

## 4️⃣ Configurar variáveis de ambiente

```bash
cp .env.example .env.local
```

---

## 5️⃣ Executar aplicação

```bash
npm run dev
```

---

## 6️⃣ Abrir no navegador

```txt
http://localhost:3000
```
