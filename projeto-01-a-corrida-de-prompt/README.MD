# 📋 Descrição do Projeto

Este projeto documenta uma atividade prática de **Engenharia de Prompt** focada na construção, refinamento e otimização de prompts para modelos de IA generativa.

A proposta consistiu em resolver dois desafios distintos:

- ✉️ Geração narrativa textual contextualizada
- 🎨 Geração imagética artística controlada

Durante o processo foram aplicadas técnicas de:

- **Iterative Prompt Refinement**
- **Chain-of-Thought Prompting**
- **Context Injection**
- **Role Prompting**
- **Constraint-Based Prompting**
- **Semantic Reinforcement**
- **Output Steering**

O objetivo principal foi reduzir respostas genéricas e aumentar a precisão semântica da IA através de refinamentos progressivos.
